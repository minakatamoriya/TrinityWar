# TrinityWar 阵营差异化首发方案 v0.1

## 日期

2026-05-27

## 一、目标

在不扩展复杂系统规则的前提下，先明确三界在首发版本中的核心方向差异，让玩家在第一次选择阵营时能够理解：

1. 人界更适合种田
2. 仙界更适合养宠
3. 魔界更适合战斗

本方案当前只做：

1. 方向定义
2. 数值方案记录
3. 新手选阵营文案收口

本方案当前不做：

1. 后端数值接入
2. 战斗公式联动
3. 阵营成长线深度扩展

## 二、首发定位

### 人界

定位：更适合种田与长期经营

玩家心智：

- 灵田更稳
- 收菜体验更顺
- 适合把经营节奏走扎实

首发方向：

- 丰熟窗口更长
- 成熟收益更高

当前预定数值：

- 丰熟窗口 +20%
- 成熟收益 +5%

### 仙界

定位：更适合养宠与成长推进

玩家心智：

- 灵宠成长更快
- 投喂收益更高
- 更适合把主宠持续养起来

首发方向：

- 挂机经验更高
- 投喂加速更久

当前预定数值：

- 挂机经验 +10%
- 投喂加速时长 +25%

### 魔界

定位：更适合战斗与主动出击

玩家心智：

- 出手更狠
- 连战续航更强
- 更适合掠夺与压制节奏

首发方向：

- 战斗伤害更高
- 战后恢复更强

当前预定数值：

- 战斗伤害 +5%
- 战后恢复已损生命 20%

## 三、新手选阵营文案原则

新手第一次选择阵营时：

1. 不展示详细数值
2. 不解释复杂公式
3. 只说明每个阵营在哪个方向更强

建议文案口径：

- 人界：更适合种田经营
- 仙界：更适合灵宠成长
- 魔界：更适合主动战斗

## 四、当前结论

阵营差异化在首发阶段先作为：

1. 新手选阵营时的方向认知
2. 后续数值接入的预留设计

当前先不做复杂延展，例如：

1. 阵营专属掉落
2. 阵营专属战斗事件
3. 阵营专属灵宠池
4. 多层阵营克制关系

后续如果继续推进，优先顺序应为：

1. 接入基础数值
2. 补充首页/阵营页说明
3. 再考虑更深层联动
