# 2026-06-22 灵植开放与灵宠合成时间表 v0.1

## 1. 文档目的

本文用于把当前版本中与`灵植开放`、`灵宠合成`有关的最新设定、节奏问题和建议时间表整理成一份统一口径，方便后续继续讨论数值与机制。

本版重点不是重写整套系统，而是回答两个问题：

1. 现在的开放/合成时间是否过慢或过快。
2. 在不让玩家太快看穿全部内容的前提下，怎样让玩家尽早建立“我拿得到”的确定感。

---

## 2. 当前最新设定核对

以下内容以当前已接入的策划文档和服务端实现为准。

### 2.1 灵植

当前灵植侧已经不是碎片制，而是`永久资格解锁`。

现行口径：

1. 新玩家默认开放 4 块田。
2. 开局已有基础灵植资格：教程种 `启灵芽`，常规基础种 `青灵麦`、`风云稻`。
3. 灵植解锁后可永久播种，不消耗库存，不走灵植碎片。
4. 灵植图鉴状态为 `undiscovered / discovered / ready / unlocked`。

当前实际门槛：

| 档位 | 当前门槛 |
|---|---|
| 基础灵植 | 直接开放 |
| 普通灵植第 1 档 | 累计收获 6 次 |
| 普通灵植第 2 档 | 累计收获 12 次 |
| 普通灵植第 3 档 | 累计收获 18 次 |
| 普通灵植第 4 档 | 累计收获 24 次 |
| 普通灵植第 5 档 | 累计收获 30 次 |
| 稀有灵植 | 阵营贡献 100 |
| 传说灵植 | 阵营贡献 200 |

这意味着：

1. 普通灵植目前是按收获次数分 5 档开放。
2. 稀有和传说灵植目前是按贡献值开放。
3. 普通灵植的批量开放问题已经开始收口，但后续仍可继续微调各档间距。
4. 当前还没有把“首次解锁”和“新赛季激活”明确拆开。

### 2.2 灵宠

当前灵宠侧已经是三段结构：

1. `未发现`
2. `发现并解锁图鉴`
3. `合成拥有`

现行口径：

1. 新玩家默认`没有直接拥有首宠`。
2. 3 只新手灵宠会先进入`可见`状态，首宠合成走新手赠送口径。
3. 非新手赠送灵宠按精魄门槛合成。
4. 灵宠图鉴状态为 `hidden / visible-progress / unlocked`。

当前实际门槛：

| 稀有度 | 合成所需精魄 |
|---|---:|
| 普通 | 20 |
| 稀有 | 60 |
| 传说 | 100 |

### 2.3 当前已接入的主要精魄来源

本轮核对到的稳定来源主要有：

1. 新手首宠赠送合成。
2. 掠夺/战斗掉落精魄。
3. 阵营每日俸禄。
4. 赛季奖励。
5. 天机符商店当前只卖灵根、灵髓、灵玉、兽魂，不直接卖灵宠精魄。

当前实现中的关键细节：

1. `第一次阵营俸禄不会发灵宠精魄`，只从第二次起才进入正常精魄奖励。
2. 当前俸禄中的精魄池是`普通非首发灵宠池`。
3. 当前赛季奖励中的精魄也落在`普通非首发灵宠池`。
4. 当前代码里，`稀有/传说精魄`缺少稳定的非战斗保底来源，主要还是靠战斗掉落。
5. 当前商业入口只保留`天机符`，没有第二套商业券体系。
6. 签到当前不作为灵宠精魄投放来源，仍以天机符等资源为主。

### 2.4 当前战斗掉落强度

当前战斗精魄掉落率是 100%，但数量按守方稀有度和胜利档位区分：

| 守方稀有度 | 小胜 | 大胜 | 完胜 |
|---|---:|---:|---:|
| 普通 | 2 | 3 | 4 |
| 稀有 | 1 | 2 | 3 |
| 传说 | 1 | 1 | 2 |

这套规则意味着：

1. 普通灵宠的首只合成可以做得比较快。
2. 稀有和传说如果没有定向来源，会明显更看对手池和战斗频次。

### 2.5 当前商业化边界

当前版本里，商业化资源的主入口是`天机符`。

现行已接入能力：

1. 天机符商店可购买灵根、灵髓、灵玉、普通/稀有/传说兽魂。
2. 天机符也已经用于部分功能加速、窥视、词条锁定等商业便利点。
3. 当前`没有直接出售灵宠碎片`的正式入口。

### 2.6 当前真实贡献来源

本节只记录当前代码里`已经正式生效`的贡献来源，不包含旧草案、已停用任务或已退役入口。

当前正式生效：

| 来源 | 当前数值 | 说明 |
|---|---:|---|
| 开始一轮非教程种植 | 1 | 非教程灵植开始培育时发放 |
| 收取一次灵植 | 2 | 非教程种的正常收取会发放 |
| 掠夺结算 | 2 | 每次有效掠夺结算都会发放 |
| 掠夺胜利追加 | 3 | 只有胜利时追加发放 |
| 灵宠洗练 | 3 | 两种洗练路径都会发放 |
| 好友采摘 | 2 | 社交助力贡献 |
| 好友复活 | 1 | 社交助力贡献 |

当前已停用或不生效：

1. 灵宠恢复贡献：当前灵宠战后自动恢复，这条贡献来源可视为废弃。
2. 冲突对抗 / 成功掠夺任务：相关任务默认关闭，不作为当前正式来源。
3. 旧版阵营日常上缴：已停用。
4. 阵营捐献：已停用。

当前赛季规则：

1. 个人阵营贡献会在赛季切换时清零。

## 3. 当前节奏问题

### 3.1 灵植问题

当前灵植的核心问题不是太慢，而是`同档位成批开放`。

具体表现：

1. 旧版普通灵植曾出现 10 次、20 次、30 次几个大包节点，本轮已收口为更平滑的 6/12/18/24/30。
2. 稀有与传说既然准备按同批贡献开放，就必须避免同批内部出现明显上下位。
3. 高阶灵植还需要一套简单的赛季激活口径，避免新赛季开局直接全员种最强植物。

这会导致两个风险：

1. 普通层不是“持续发现新东西”，而是“到一个点突然看穿一批东西”。
2. 高阶层如果同批开放但内部强弱分化明显，玩家会直接只选一两个公认最强解。

除此之外，灵植还有一个赛季制问题：

1. 如果高阶灵植永久可种，新赛季开局很容易直接跳到最强植物。
2. 如果高阶灵植每赛季完全重做不同门槛，又会让规则变复杂、重复劳动感变强。

所以灵植侧需要一个`简单、统一、可继承`的赛季口径。

### 3.2 灵宠问题

当前灵宠的核心问题不是三段结构，而是`首次可感知时间`与`高阶确定性来源`。

具体表现：

1. 首日如果只靠俸禄，玩家拿不到精魄感知。
2. 普通灵宠有俸禄和赛季奖励兜底，问题相对小。
3. 稀有/传说灵宠目前更依赖战斗掉落，稳定感不够。
4. 如果不给目标追踪，玩家容易出现“碎片很多，但没有一只快成”的体验。

### 3.3 掉落归属问题

如果灵宠碎片完全采用“打谁掉谁”的纯敌方掉落逻辑，会有两个问题：

1. 节奏控制权过度依赖目标池、刷新和运气。
2. 稀有/传说的可见频率一旦不足，玩家就会卡在“知道有，但就是凑不齐”。

如果灵宠碎片完全采用“系统发什么就拿什么”的纯系统发放逻辑，也会有两个问题：

1. 战斗对象本身的辨识价值下降。
2. 玩家容易觉得“打谁都差不多”，狩猎感变弱。

因此更推荐采用`混合制`：

1. 普通战斗负责掉`敌方同种碎片`，保留直觉和狩猎感。
2. 俸禄、赛季奖励和其他系统奖励负责`系统定向发放`，保留节奏控制。
3. 如果后续重新开放日常/周常奖励，也应归到系统定向发放层，而不是回到纯随机层。

当前更明确的投放分工：

1. `签到`：不发灵宠碎片，继续承担天机符和其他轻资源投放。
2. `签到阶段宝箱`：承担低频强奖励，可联动稀有/传说碎片。
3. `俸禄`：承担前期最稳定的灵宠碎片来源。
4. `战斗`：承担敌方同种掉落与目标感。
5. `赛季奖励`：承担中后段补强与收尾。

### 3.4 商业券直接卖碎片的问题

如果直接开放“商业券直接购买目标灵宠碎片”，会带来三个明显风险：

1. 战斗掉落的意义被削弱，玩家更容易从“去打目标”转成“直接买目标”。
2. 碎片制的时间表会被商业券直接穿透，稀有/传说的首只完成时间会明显失控。
3. 玩家会更快把“图鉴收集”理解成付费通道，而不是经营与战斗通道。

所以这个问题不是“能不能卖”，而是“卖到什么程度不会打穿主循环”。

### 3.5 当前贡献数值与灵植门槛不匹配

按旧口径来看，`300 / 800` 这组高阶灵植贡献门槛偏高。

原因不是概念问题，而是当前正式贡献来源总量偏低：

1. 收菜每次只有 `+2`。
2. 如果种菜不单独给贡献，新玩家前期增长会偏慢。
3. 掠夺即使改成“结算 +2，胜利额外 +3”，普通玩家的贡献增长也依然有限。
4. 已停用的任务、上缴、捐献都不能再补贡献。

这意味着：

1. `800` 对普通玩家明显过高。
2. `300` 对不那么频繁上线的玩家也偏紧。
3. 因此高阶灵植门槛更适合下调到 `100 / 200`。

## 4. 节奏目标

本版建议采用下面这条总原则：

1. `看见目标`要早。
2. `第一次拿到进度`要早。
3. `第一次真正完成`要适度靠后。
4. `全收集`必须显著慢于`第一次成型`。

换成玩家感受就是：

1. 我知道后面有东西。
2. 我已经在攒，不是摆设。
3. 我这周内能拿到一点成果。
4. 但后面还有很多内容没被我吃完。

## 5. 灵植开放时间表

### 5.1 建议目标时间表

| 时间 | 建议状态 | 设计目的 |
|---|---|---|
| 第 0 天 | 教程种 + 基础常规种开放 | 立刻进入种田主循环 |
| 第 2-3 天 | 解锁第 1 个普通灵植 | 建立“种田会持续开新植物”的感知 |
| 第 3-5 天 | 解锁第 2 个普通灵植 | 保持前期探索密度 |
| 第 5-7 天 | 解锁第 3 个普通灵植 | 首周形成明确成长感 |
| 第 7-10 天 | 解锁第 4-5 个普通灵植 | 普通层基本铺开，但不一次性看穿 |
| 第 10-14 天 | 同批解锁全部稀有灵植 | 第 2 周给出一次明确的高阶选择面 |
| 第 17-21 天 | 同批解锁全部传说灵植 | 第 3 周给出一次明确的终局选择面 |
| 第 21 天后 | 进入高阶灵植使用、比较与阵容化阶段 | 承接赛季后段经营与选择深度 |

### 5.2 对当前门槛的建议

如果继续沿用“普通看收获，稀有/传说看贡献”的大框架，建议做以下调整：

1. 普通灵植继续保留平滑的分档门槛，不回退到`10/20/30`这种大包式门槛。
2. 稀有和传说既然已经决定按同批贡献开放，就不要再额外拆小门槛。
3. 灵植应改成：`普通分批开放，高阶按稀有度整批开放`。

更推荐的拆法：

| 层级 | 当前问题 | 建议方向 |
|---|---|---|
| 普通 | 10、20、30 会成批解锁 | 拆成 8-10、15-18、22-25、30-32、36-40 |
| 稀有 | 同批开放时容易出现内部唯一最优解 | 统一放在 100 贡献，同时尽量做并列选择 |
| 传说 | 同批开放时容易出现内部唯一最优解 | 统一放在 200 贡献，同时尽量做并列选择 |

### 5.3 灵植机制建议

为了兼顾探索欲，灵植建议采用：

1. `发现`和`正式解锁`分离。
2. 前 7 天普通灵植持续解锁。
3. 第 2 周整批开放稀有，第 3 周整批开放传说。
4. 高阶灵植可以提前以图鉴、情报或预览方式让玩家知道存在，但正式种植资格按贡献解锁。

这样做的好处是：

1. 首周有持续新鲜感。
2. 高阶层规则简单，不会出现一堆难记的小门槛。
3. 玩家会形成“第 2 周碰稀有，第 3 周碰传说”的稳定预期。

### 5.4 灵植赛季激活规则建议

本版建议把高阶灵植的赛季规则收口到最简单的一套：

1. `基础灵植、普通灵植`：一旦解锁，后续赛季直接可种，不再重锁。
2. `稀有灵植、传说灵植`：图鉴与收藏永久保留，但每个新赛季开始后，需要本赛季贡献再次达到对应门槛，才恢复种植资格。
3. `首次解锁门槛 = 次赛季激活门槛`，不额外设计第二套数值。
4. 不增加周限制，不增加额外任务限制，只看贡献值。

这套规则的优点：

1. 简单，玩家最好理解。
2. 不会让老玩家在新赛季开局直接全种最强植物。
3. 不会让玩家觉得“上赛季白解锁了”，因为图鉴、收藏和已获得感都保留。
4. 前后端展示、赛季重置和数值维护都更轻。

建议文案：

1. `已解锁`
2. `本赛季未激活`
3. `本赛季已激活`

不要把赛季激活继续叫“重新解锁”，避免玩家误会。

### 5.5 结合当前设定的贡献门槛表

如果目标是：

1. 不那么频繁上线的玩家也能在`第 10-14 天`解锁首个稀有灵植。
2. 不那么频繁上线的玩家也能在`第 17-21 天`解锁首个传说灵植。
3. 规则继续沿用当前贡献体系，不额外造新节点。
4. 并默认会把掠夺贡献改成“结算固定 +2，胜利额外 +3”，而不是继续维持“只有胜利才给贡献”。

那么建议直接定成：

| 类型 | 首次解锁门槛 | 次赛季激活门槛 | 建议达到时间 |
|---|---:|---:|---|
| 全部稀有灵植 | 100 贡献 | 100 贡献 | 赛季第 10-14 天 |
| 全部传说灵植 | 200 贡献 | 200 贡献 | 赛季第 17-21 天 |

这套节奏的含义是：

1. 轻度或不频繁上线玩家，第 2 周看到整批稀有植物可用，希望明确且不算太晚。
2. 第 3 周看到整批传说植物可用，既不会开局太快，也不会拖到赛季快结束。
3. 老玩家下赛季也还是要重新经营贡献，但规则完全不变，理解成本极低。

### 5.6 对当前灵植门槛的直接建议

基于你现在的设定，我更推荐的收口方式是：

1. 基础灵植：直接开放。
2. 普通灵植：继续按收获次数开放，但拆细节点，不再成批。
3. 全部稀有灵植：100 贡献，同批解锁，同批激活。
4. 全部传说灵植：200 贡献，同批解锁，同批激活。
5. 不再额外拆分第二批稀有、第二批传说的贡献节点。

一句话收口：

> 高阶灵植只看贡献，稀有同批 100、传说同批 200；首次解锁和次赛季激活同门槛，普通保永久可种，高阶保永久收藏 + 赛季贡献激活。

## 6. 灵宠合成时间表

### 6.1 需要拆开的三个时间点

灵宠节奏不要只看“多久合成”，而要拆成：

1. `T1`：第一次拿到该档位精魄。
2. `T2`：玩家确信自己可以靠积累合成。
3. `T3`：真正完成合成。

本版建议是：

1. `T1` 很早。
2. `T2` 不拖。
3. `T3` 适度靠后。

### 6.2 建议目标时间表

| 时间 | 建议状态 | 设计目的 |
|---|---|---|
| 第 0 天 | 合成新手首宠 | 先把灵宠系统接上，不让玩家空等 |
| 第 1 天 | 拿到首个非新手灵宠精魄 | 建立“精魄是可积累资产”的第一印象 |
| 第 2-3 天 | 再次拿到同目标精魄 | 让玩家确认不是一次性展示 |
| 第 4-7 天 | 合成首只普通非新手灵宠 | 给出首个碎片制闭环成果 |
| 第 5-7 天 | 看到首个稀有精魄进度 | 提前建立中期目标 |
| 第 7 天前 | 看到首个传说精魄进度 | 让高阶目标进入视野，但不立刻拿满 |
| 第 10-14 天 | 合成首只稀有灵宠 | 进入阵容选择阶段 |
| 第 14-21 天 | 合成首只传说灵宠 | 首赛季后段形成高价值成果 |
| 第二赛季 | 第 2 只传说或更深培养 | 承接长期追求 |

### 6.3 对当前来源的判断

当前来源结构下：

1. 首宠教程节奏是成立的。
2. 普通灵宠的节奏基本可以通过战斗 + 俸禄 + 赛季奖励做出来。
3. 稀有/传说灵宠还缺少足够稳定的非随机来源。

所以当前最需要补的不是“更高掉率”，而是：

1. 首日首片的确定来源。
2. 稀有/传说的定向积累来源。
3. 非纯随机的保底进度。

### 6.4 灵宠机制建议

#### 1. 保留首宠赠送，避免首日系统空转

当前新手首宠走赠送合成是合理的，不建议撤掉。

原因：

1. 它负责把灵宠页、五行选择、主位战斗快速串起来。
2. 它不等于后续高阶灵宠都很快。

#### 2. 首日首片不要依赖第一次俸禄

因为当前实现里第一次阵营俸禄不发精魄，所以首日首片必须由下面来源兜住：

1. 新手首战固定掉片。
2. 首次探索固定掉片。
3. 新手任务保底给片。

只要首日没首片，玩家就容易把“灵宠精魄”理解成遥远内容。

#### 2.1 签到阶段宝箱建议

日签到本身不发灵宠碎片，但阶段宝箱可以承担`低频高价值`奖励。

当前建议节点：

1. `1天`
2. `2天`
3. `3天`
4. `5天`
5. `7天`
6. `14天`
7. `21天`

建议定位：

1. `1天 / 2天 / 3天`：偏福利启动
2. `5天 / 7天`：开始明显推动普通与稀有碎片进度
3. `14天 / 21天`：明显推动传说碎片进度

测试阶段口径：

1. 所有阶段宝箱都可点击领取，用于快速验证奖励与入账闭环
2. 正式阶段再收口到“达成对应签到天数才可领取”

#### 3. 稳定来源和随机来源分工

建议把精魄来源拆成两类：

1. `稳定来源`：负责让玩家相信最终能合成。
2. `随机来源`：负责惊喜感和波动。

推荐比例：

1. 首只普通灵宠：70% 稳定，30% 随机。
2. 首只稀有灵宠：60% 稳定，40% 随机。
3. 首只传说灵宠：至少 50% 稳定，不建议纯随机。

#### 4. 普通战斗与系统奖励的分工

本版更推荐采用下面这套明确分工：

1. `普通战斗`：稳定掉落敌方同种碎片，不掉其他灵宠碎片。
2. `俸禄 / 赛季奖励 / 其他系统奖励`：偏系统定向发放，用来补玩家追踪目标进度。
3. `如果后续重开日常/周常奖励`：也应走系统定向发放，而不是回到纯随机掉落。

其中普通战斗不建议“输赢都掉同种碎片”，但建议`输也给少量参与贡献`。

更推荐的结算方式：

| 战斗结果 | 建议奖励 |
|---|---|
| 完胜 | 敌方同种碎片高档掉落 |
| 大胜 | 敌方同种碎片中档掉落 |
| 小胜 | 敌方同种碎片低档掉落 |
| 平局 | 不掉敌方同种碎片，给少量通用补偿 |
| 失败 | 不掉敌方同种碎片，给少量通用补偿 |

这样设计的原因：

1. 战斗对象有意义，打熊主要拿熊碎片。
2. 战斗结果也有意义，不会变成“碰一下就稳定拿进度”。
3. 真正的收集节奏仍然由系统奖励层兜底，不会完全失控。

这里的`通用补偿`更建议是：

1. 灵根
2. 兽魂
3. 通用积分/灵魄券

不建议失败也直接稳定发目标碎片。

#### 4.1 掠夺贡献建议

为了让普通玩家的贡献曲线更稳定，也为了避免“打输了完全白打”，建议把掠夺贡献拆成两段：

1. `每次有效掠夺结算固定 +2 贡献`
2. `若战斗胜利，再额外 +3 贡献`

这样收口后：

| 掠夺结果 | 建议贡献 |
|---|---:|
| 胜利 | 5 |
| 失败 | 2 |

这套设计的优点：

1. 失败不至于完全没有进度，普通玩家更敢尝试掠夺。
2. 胜利仍然明显更值钱，不会抹平战斗结果。
3. 有利于把高阶灵植门槛控制在 `100 / 200` 这种更贴近当前生态的区间。

#### 4.2 新玩家贡献底盘建议

为了照顾新玩家前期：

1. 没有太多好友。
2. 掠夺可能会输。
3. 灵宠还没进入洗练阶段。

贡献底盘应该建立在`种菜 + 收菜`这条最稳定的循环上。

建议口径：

| 行为 | 建议贡献 |
|---|---:|
| 开始一轮非教程种植 | 1 |
| 收取一次非教程灵植 | 2 |
| 掠夺结算 | 2 |
| 掠夺胜利追加 | 3 |
| 灵宠洗练 | 3 |
| 好友采摘 | 2 |
| 好友复活 | 1 |

这套结构的含义是：

1. 农场循环提供基础贡献。
2. 掠夺、洗练、社交提供加速贡献。
3. 就算新玩家前期几乎不靠好友、不靠洗练，也能靠自己稳定推进高阶灵植激活。

#### 5. 增加目标追踪

如果不做目标追踪，当前俸禄和赛季奖励的随机普通池会把碎片打散。

建议增加：

1. 玩家可选择 1 只追踪灵宠。
2. 稳定来源优先掉该目标。
3. 随机来源仍保留小概率打散，维持惊喜。

#### 6. 高阶灵宠建议“先见到，后合成”

对于稀有和传说，建议采用：

1. 第一次精魄出现时间前置。
2. 前 7 天做到 10%-30% 可见进度。
3. 真正合成放到第 10-21 天区间。

这样既不会太慢，也不会太快吃穿。

### 6.5 商业券与碎片购买建议

#### 1. 不建议直接卖目标碎片

首发阶段不建议开放：

1. 用天机符直接购买指定灵宠碎片。
2. 用天机符无限量购买随机灵宠碎片。
3. 用商业券直接补满稀有/传说合成缺口。

原因：

1. 会直接打穿灵宠战斗掉落的意义。
2. 会让碎片制过快变成“付费拼图”。
3. 会明显压缩你原本最需要控制的 7 天、14 天、赛季内节奏。

#### 2. 更适合卖的是“追赶效率”，不是“结果本身”

如果要开放商业券相关入口，更推荐卖：

1. 额外挑战次数。
2. 额外目标刷新次数。
3. 目标追踪位或追踪加成。
4. 通用补偿资源，如灵根、兽魂、灵髓、灵玉。
5. 少量`碎片代币/灵魄券`，再通过周限购或兑换上限慢慢换。

也就是：

> 商业券更适合卖“多打几次、补一点效率、少走一点弯路”，不适合直接卖“这只宠的最终碎片结果”。

#### 3. 如果一定要卖碎片，建议只开弱入口

如果后续确实要试商业券买碎片，建议只开下面这种弱入口：

1. 只卖`随机普通碎片包`，不卖指定碎片。
2. 只卖`通用碎片代币`，且有严格周限购。
3. 稀有/传说碎片不直接卖，只能通过高成本、低频率的兑换慢慢补尾差。
4. 不允许玩家靠商业券在 1-3 天内直接完成首只传说。

换句话说：

1. 可以卖`补尾差`
2. 不要卖`跳阶段`

## 7. 现阶段最值得改的点

如果这一轮只改最关键的地方，优先级建议如下：

### P1. 先改灵植的“批量开放”

原因：

1. 这是当前最容易破坏探索欲的点。
2. 改完后，灵植会从“到点一锅端”变成“持续有新目标”。

### P2. 先补灵宠首日首片来源

原因：

1. 第一次阵营俸禄不发精魄。
2. 所以首日首片必须由教程或首战兜底。

### P3. 再补稀有/传说灵宠的稳定来源

原因：

1. 目前普通灵宠已有较多自然来源。
2. 真正容易“遥遥无期”的是高阶灵宠。

### P4. 再定战斗掉落和系统奖励的分工

原因：

1. 这是控制“狩猎感”和“可控节奏”平衡的关键。
2. 先把战斗只掉敌方同种碎片、系统奖励负责定向补进度、掠夺失败也给少量参与贡献的口径定下来，后续数值才好调。

### P5. 最后补目标追踪、阶段奖励和商业化边界

原因：

1. 这一步是为了解决“有进度但没有归属感”。
2. 能把“随机掉了一堆”转成“我这只快成了”。
3. 也能避免商业券入口一上来就把碎片节奏打穿。

## 8. 本版结论

### 8.1 灵植

灵植现在的主要问题不是太慢，而是`同层批量开放`会破坏探索节奏。

建议：

1. 保留“基础开放 + 收获/贡献解锁”的大框架。
2. 改成阶梯式单个开放。
3. 让普通灵植在首周持续释放，稀有在第 2 周同批出现，传说在第 3 周同批出现。

### 8.2 灵宠

灵宠现在的主要问题不是三段结构，而是`首日感知`和`高阶确定性来源`。

建议：

1. 保留新手首宠赠送合成。
2. 首日必须给首个非新手精魄感知。
3. 普通战斗主要掉`敌方同种碎片`，但只在胜利时发放。
4. 平局和失败只给少量通用补偿，不直接稳定给目标碎片。
5. 掠夺贡献建议改成“结算固定 +2，胜利额外 +3”，让失败也有少量参与进度。
6. 俸禄、赛季奖励和其他系统奖励负责系统定向补进度。
7. 如果后续重开日常/周常奖励，也应归入系统定向补进度层。
8. 普通灵宠在 4-7 天完成首只非新手合成。
9. 稀有灵宠放在 10-14 天完成首只。
10. 传说灵宠在首周进入视野，在 14-21 天完成首只。

### 8.3 商业券

商业券当前更适合卖`效率`，不适合直接卖`目标灵宠碎片结果`。

建议：

1. 首发不直接开放指定灵宠碎片购买。
2. 优先开放挑战次数、刷新次数、追踪加成、培养资源等弱穿透入口。
3. 如果后续一定要卖碎片，也只卖低强度、低频率、不能跳阶段的弱入口。

一句话总结：

> 该前置的是“希望感”和“可积累感”，不该前置的是“全收集”和“全看穿”。
