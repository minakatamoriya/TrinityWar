好的，基于之前的讨论，我把 **“新特质”** 和 **“条件增益词条”** 整合成一版完整的方案。这套方案严格遵循你定下的原则：

- **特质 = 宠物的“战术标签”**（改变节奏/结算规则，有辨识度）
- **纯增益词条 = 主力（60%）**（无脑爽，不费脑子）
- **条件增益词条 = 少数（20%）**（有触发门槛，但没有负面代价）
- **高风险词条 = 极少数（20%）**（有明显代价，给深度玩家）

## 一、特质方案（宠物的“灵魂标签”）

以下特质全部满足：**一眼看懂、不改战斗结构、围绕“攻击/血量/结算”做文章**。

我按“打法风格”分了4类，方便你后续给宠物贴标签：

### 1. 时间轴型（什么时候强）

| 特质名 | 效果（玩家感知一句版） | 策略定位 |
| :--- | :--- | :--- |
| **蓄势** | 前3回合攻击-50%，第4回合起攻击永久+40% | 憋大招，前期挨打后期一拳超人 |
| **突袭** | 前3回合攻击+20% | 开局压制，抢血线优势 |
| **后劲** | 第6回合起，攻击每回合+8%（最多叠加5次） | 越打越疼，后期翻盘 |
| **疲软** | 前5回合攻击+15%，第6回合起攻击-15% | 前期猛冲，后期乏力（适合赌前5回合打崩对手） |

### 2. 血线型（什么血量时强）

| 特质名 | 效果（玩家感知一句版） | 策略定位 |
| :--- | :--- | :--- |
| **破釜** | 血量首次低于40%时，攻击翻倍（持续到结束） | 残血战神，极限翻盘 |
| **恃强** | 血量高于对手时，攻击+20%；低于对手时，攻击-10% | 滚雪球，优势时更强 |
| **同命** | 每回合结束，若自己血量百分比低于对手，则拉平至双方平均值 | 拖人下水，克制高爆发 |
| **金蝉** | 血量每损失10%，攻击+6%（最多+30%） | 越残越凶，主动卖血换输出 |

### 3. 结算型（10回合结束时动手脚）

| 特质名 | 效果（玩家感知一句版） | 策略定位 |
| :--- | :--- | :--- |
| **逆转** | 第10回合结算时，若自己血量低于对手，则追加“本场总伤害×20%”的真实伤害 | 靠总伤害翻盘，鼓励堆攻击 |
| **偷鸡** | 第10回合结算时，若双方血量差值小于5%，则判自己胜 | 焦灼局专杀，制造“差一点就赢”的爽感 |
| **铁秤** | 第10回合结算时，若双方血量差值小于10%，则判自己胜（比偷鸡更宽松） | 同上，但更强（适合传说宠） |
| **同归** | 第10回合结算时，若双方血量差值小于8%，则双方各扣当前血量的30%后重新判定 | 焦灼局重新洗牌，增加戏剧性 |

### 4. 特殊机制型（独一份的歪门邪道）

| 特质名 | 效果（玩家感知一句版） | 策略定位 |
| :--- | :--- | :--- |
| **燃血** | 前5回合，每回合自损当前血量的8%，每自损一次攻击永久+15%（最多5次） | 主动卖血换极致输出，风险最高回报最大 |
| **均富** | 第3回合结束，若自己血量高于对手，则转移10%血量给对手，攻击永久+15% | 主动削自己换攻击，同时拉近血线 |
| **乞讨** | 第5回合结束，若自己血量低于对手，则偷取对手当前血量的8%加给自己，攻击永久+10% | 逆风偷血偷攻，翻盘神器 |

## 二、词条方案（通用槽位）

**5个槽位**，玩家自由搭配。以下按“纯增益 → 条件增益 → 高风险”三层结构列出：

### 第一层：纯增益（约60%，无脑装）

| 词条名 | 效果（玩家一眼看懂） | 定位 |
| :--- | :--- | :--- |
| **利爪** | 攻击 +10% | 无脑堆攻 |
| **厚皮** | 生命 +10% | 无脑堆血 |
| **暴击** | 暴击率 +8% | 赌狗快乐词条 |
| **爆伤** | 暴击伤害 +25% | 暴击流核心 |
| **闪避** | 闪避率 +6% | 生存赌狗词条 |
| **吸血** | 造成伤害的10%回复自身 | 续航万金油 |

### 第二层：条件增益（约20%，无负面，有门槛）

| 词条名 | 效果（玩家一眼看懂） | 触发条件 | 定位 |
| :--- | :--- | :--- | :--- |
| **背水** | 攻击 +25% | 自身血量低于50%时生效 | 残血反杀，配合“破釜/金蝉”特质 |
| **欺软** | 伤害 +18% | 目标血量高于自己时生效 | 专打坦克，克制堆血流 |
| **压制** | 伤害 +18% | 目标血量低于自己时生效 | 滚雪球，优势时更强 |
| **狂战** | 本次攻击伤害×1.5倍 | 每次攻击有20%概率触发 | 纯概率爽感，触发时数字跳大字 |
| **虚弱** | 目标下回合攻击-20% | 每次攻击有20%概率触发 | 软控制，不改变结构但影响后续 |

### 第三层：高风险高回报（约20%，有明显代价）

| 词条名 | 效果（玩家一眼看懂） | 代价 | 定位 |
| :--- | :--- | :--- | :--- |
| **利刃** | 攻击 +30% | 生命上限 -10% | 玻璃大炮，极致输出 |
| **铁骨** | 生命 +30% | 攻击 -10% | 极致血牛，拖到结算 |
| **贪婪** | 吸血 +20% | 受到的伤害 +10% | 极端续航，吸得猛但挨打也疼 |
| **鲁莽** | 暴击率 +15% | 暴击伤害 -30% | 高频低伤暴击，配合“连击”类特质 |

> **UI提示**：第三层词条卡片右下角加一个 **红色⚠️角标**，让玩家一眼知道“这条有副作用，新手慎用”。

## 四、稀有度与特质/词条的关系

| 稀有度 | 特质数量 | 词条槽位 | 特质强度 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| **普通** | 1条 | 5槽 | 基础数值（如突袭+15%） | 身份清晰，上手即用 |
| **稀有** | 1条 | 5槽 | 数值更高（如突袭+22%） | 同款特质的升级版，更稳更强 |
| **传说** | 2条 | 5槽 | 两条特质组合 | 比如“蓄势+破釜”，前3挨打、残血翻倍，双重保险 |

**注意**：词条槽位所有稀有度**统一都是5个**，稀有度的差异只体现在“特质条数”和“特质数值”上。这样玩家抽到普通宠也不会觉得“词条少一块”，只是“灵魂标签”弱一点而已。

## 五、这套方案的三大优势

1. **上手门槛极低**：60%的词条是纯增益，新手无脑装“利爪+厚皮+暴击”就能玩。条件增益只占少数，且没有负面代价，玩家不需要计算“装了会不会变弱”。
2. **深度足够**：传说宠2条特质的组合（如“蓄势+逆转”）、高风险词条的取舍（“我要不要为了30%攻击牺牲10%血量？”）、条件词条的触发时机（“我该堆血触发背水，还是堆攻触发压制？”）——足够老玩家研究很久。
3. **完全适配“10回合看血量”**：所有特质和词条都围绕“攻击”和“血量”展开，没有任何防御/护甲/破甲等需要理解公式的概念。结算型特质（偷鸡、逆转、铁秤）更是直接针对“10回合看血量”这个胜负规则做文章，让每场对局的悬念保持到最后一刻。
