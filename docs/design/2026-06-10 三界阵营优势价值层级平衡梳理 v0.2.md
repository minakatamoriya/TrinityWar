# 2026-06-10 三界阵营优势价值层级平衡梳理 v0.2

## 一、文档目标

本文档用于替代 v0.1 中较分散的方案讨论，整理为可直接进入实现、配置和 28 天赛季模拟的 v0.2 口径。

v0.2 的核心目标不是推翻“三界分工”，而是让三界优势处在更接近的价值层级：

- 人界：稳定经营，金币更稳，洗练更省，被抢更少。
- 仙界：高效修行，灵根更多，经验更快，防守更稳。
- 魔界：主动进攻，成熟更快，突破更省，主动攻击更强。

本版重点解决 v0.1 诊断出的核心问题：

```text
人界/仙界优势偏单点、偏线性。
魔界优势偏闭环、偏复合。
```

因此 v0.2 不再让魔界同时拥有攻击、收益、续航和防守外溢优势，而是改为：

```text
魔界主动进攻更强，但连续战斗不再无成本回血。
```

## 二、v0.2 平衡原则

### 1. 三界都必须覆盖三大领域

每个阵营都在以下三个领域拥有至少一个明确优势：

- 种田。
- 养宠。
- 战斗。

这样可以避免某个阵营只在一个玩法中有存在感，也避免某个阵营的优势完全被对手核心玩法吞掉。

### 2. 主优势必须配一个抗压手段

每个阵营都必须有：

```text
主优势 + 抵抗对手优势的手段。
```

对应到三界：

| 阵营 | 主优势 | 抗压手段 | 目标体验 |
| --- | --- | --- | --- |
| 人界 | 农田金币与经营效率 | 防守战资源损失降低 | 产得多，也能留住一部分 |
| 仙界 | 灵根、经验、灵宠成长 | 防守战主宠更耐打 | 养得快，守得稳 |
| 魔界 | 主动 raid 与节奏推进 | 无战后回血，承担 HP 成本 | 打得凶，但不能无成本滚雪球 |

### 3. 进攻优势不自动外溢到防守

魔界的战斗加成只在主动 raid 中生效。

魔界作为防守方时，不享受主动进攻攻击加成。

这样保留魔界“善进攻”的身份，同时避免它因为战斗面板更强而自然变成防守最优解。

### 4. 攻击方收益必须与防守方损失一致

人界防守减损生效时：

- 防守方最终资源损失降低。
- 攻击方最终获得量同步降低。

不允许出现防守方少扣、攻击方照拿，避免凭空生成资源。

## 三、v0.2 三领域参数方案

| 阵营 | 种田优势 | 养宠优势 | 战斗优势 |
| --- | --- | --- | --- |
| 人界 | 成熟收获金币 `+10%` | 灵宠洗练金币消耗 `-10%` | 防守战中资源损失 `-20%` |
| 仙界 | 收菜获得灵根 `+10%` | 挂机经验 `+10%` | 防守战中主战灵宠最大 HP `+5%` |
| 魔界 | 作物成熟时间 `-10%` | 灵宠突破兽魂消耗 `-10%`，向上取整 | 主动 raid 攻击 `+6%`，防守不加成，无战后回血 |

### 1. 人界

人界 v0.2 定位：

```text
稳定经营，金币更稳，洗练更省，被抢更少。
```

规则：

- 成熟收获金币 `+10%`。
- 灵宠洗练金币消耗 `-10%`。
- 防守战中最终资源损失 `-20%`。
- 防守减损生效时，攻击方获得量同步降低。

说明：

- 成熟收获金币从旧版 `+5%` 提升到 `+10%`，强化人界经营主定位。
- 洗练金币减免服务赛季中后期养宠，不直接提高战斗胜率。
- 防守减损用于对抗 raid 闭环，避免人界变成“产得多但留不住”。
- 旧版可收窗口 `+20%` 可以作为已有经营容错保留，但不作为 v0.2 三领域核心优势指标。

玩家侧表达：

```text
人界收成更高，洗练更省金币，被掠夺时损失更少，适合稳定经营。
```

### 2. 仙界

仙界 v0.2 定位：

```text
高效修行，灵根更多，经验更快，防守更稳。
```

规则：

- 收菜获得灵根 `+10%`。
- 灵宠挂机经验 `+10%`。
- 防守战中主战灵宠最大 HP `+5%`。

说明：

- 收菜灵根加成让仙界种田收益与养宠目标直接联动。
- 挂机经验保留旧版核心优势，继续服务前中期成长。
- 防守战主宠 HP `+5%` 是全赛季抗压手段，但只在防守战中生效。
- v0.2 首版不采用“全场景主宠 HP +10%”，避免仙界同时获得进攻、防守和成长三重优势。
- v0.2 首版不再强调投喂持续时间 `+25%` 作为核心优势，避免满级后价值衰减。

玩家侧表达：

```text
仙界灵根更多，灵宠成长更快，防守时主战灵宠气血更稳。
```

### 3. 魔界

魔界 v0.2 定位：

```text
主动进攻，成熟更快，突破更省，主动攻击更强。
```

规则：

- 作物成熟时间 `-10%`。
- 灵宠突破兽魂消耗 `-10%`，计算结果向上取整。
- 主动 raid 时攻击 `+6%`。
- 魔界作为防守方不享受攻击加成。
- 移除旧版战后恢复已损生命 `20%`。

说明：

- 成熟时间缩短体现魔界节奏快，但需要模拟按真实成熟时间动态收菜。
- 突破兽魂减免体现“以战养宠”的成长效率，但不增加掉落概率。
- 主动攻击加成从旧版全战斗 `+5%` 调整为主动 raid `+6%`。
- 移除战后回血后，魔界连续战斗仍然需要承担 HP 成本。
- 魔界不再同时拥有高胜率、高收益、高频率、低被抢风险和无成本续航。

玩家侧表达：

```text
魔界灵田成熟更快，主动进攻更强，灵宠突破更省兽魂。
```

## 四、关键公式口径

### 1. 人界成熟收获金币

生效条件：

- 收获方阵营为人界。
- 作物进入成熟收获结算。

公式：

```text
最终成熟金币 = floor(基础成熟金币 * 1.10)
```

暂定向下取整，保持金币产出不凭空放大过多。

### 2. 人界洗练金币减免

生效条件：

- 洗练玩家阵营为人界。
- 洗练配置中存在金币消耗。

公式：

```text
最终金币消耗 = ceil(基础金币消耗 * 0.90)
```

说明：

- 只减免金币。
- 不减免灵髓。
- 不减免灵玉。
- 不改变词条概率。

### 3. 人界防守减损

生效条件：

- raid 防守方阵营为人界。
- 本次 raid 结算产生资源损失。

公式：

```text
防守方最终损失 = floor(原始损失 * 0.80)
攻击方最终获得 = 防守方最终损失
```

说明：

- 只影响可掠夺资源损失。
- 不影响兽魂随机掉落概率。
- 不影响灵宠碎片随机掉落概率。
- 不影响战斗胜负判定。

### 4. 仙界收菜灵根加成

生效条件：

- 收获方阵营为仙界。
- 作物收获会产出灵根。

公式：

```text
最终灵根 = floor(基础灵根 * 1.10)
```

说明：

- 如果单次灵根产出是随机区间，先随机得到基础灵根，再应用阵营加成。
- 不改变随机区间本身。

### 5. 仙界挂机经验加成

生效条件：

- 灵宠所属玩家阵营为仙界。
- 结算挂机经验。

公式：

```text
最终挂机经验 = floor(基础挂机经验 * 1.10)
```

说明：

- 与其他经验加成的叠加顺序需要在实现时统一。
- v0.2 首版建议与投喂加速乘算，不直接相加。

### 6. 仙界防守主宠 HP

生效条件：

- raid 防守方阵营为仙界。
- 防守方存在主战灵宠。
- 本次战斗正在计算防守方主战灵宠最大 HP。

公式：

```text
防守主宠最大 HP = floor(基础最大 HP * 1.05)
```

说明：

- 只影响防守战。
- 只影响主战灵宠。
- 不生成护盾。
- 不改变战后实际回血。
- 不影响仙界主动进攻时的 HP。

### 7. 魔界作物成熟时间

生效条件：

- 种植方阵营为魔界。
- 计算作物 matureAt / fullMatureAt。

公式：

```text
最终成熟时间秒数 = ceil(基础成熟时间秒数 * 0.90)
```

说明：

- 该规则影响真实收菜节奏。
- 模拟机器人必须按真实 matureAt 推进，不能固定按 3 小时或每天 8 轮收菜。
- 可收窗口、枯萎时间等后续时间点需要基于调整后的成熟时间重新计算。

### 8. 魔界突破兽魂减免

生效条件：

- 突破玩家阵营为魔界。
- 突破配置中存在兽魂消耗。

公式：

```text
最终兽魂消耗 = ceil(基础兽魂消耗 * 0.90)
```

示例：

| 原消耗 | 魔界消耗 |
| --- | --- |
| 普通兽魂 `5` | 普通兽魂 `5` |
| 普通兽魂 `12` | 普通兽魂 `11` |
| 稀有兽魂 `10` | 稀有兽魂 `9` |
| 稀有兽魂 `20` | 稀有兽魂 `18` |
| 传说兽魂 `8` | 传说兽魂 `8` |

说明：

- 只影响突破消耗。
- 不影响 raid 掉落概率。
- 不影响俸禄发放。
- 不影响其他养宠材料。

### 9. 魔界主动 raid 攻击

生效条件：

- raid 攻击方阵营为魔界。
- 本次战斗为主动 raid。

公式：

```text
攻击方面板攻击 = floor(基础攻击 * 1.06)
```

说明：

- 魔界作为防守方不生效。
- 不影响非 raid 的展示面板，除非该面板明确展示“主动 raid 预览”。
- 不再有战后恢复已损生命。

## 五、与旧版阵营优势的替换关系

| 阵营 | 旧版能力 | v0.2 处理 |
| --- | --- | --- |
| 人界 | 成熟收益 `+5%` | 调整为成熟收获金币 `+10%` |
| 人界 | 可收窗口 `+20%` | 可保留为经营容错，但不计入 v0.2 核心三领域优势 |
| 仙界 | 挂机经验 `+10%` | 保留 |
| 仙界 | 投喂加速持续时间 `+25%` | 暂不作为 v0.2 核心优势，可后续评估是否保留 |
| 魔界 | 全战斗攻击 `+5%` | 调整为主动 raid 攻击 `+6%` |
| 魔界 | 战后恢复已损生命 `20%` | 移除 |

如果实现上需要保持向后兼容，建议保留旧字段但新增 v0.2 字段，不要继续用旧字段表达新规则。

## 六、与动态强弱阵营规则的关系

v0.2 三界天赋与动态强弱阵营规则是两套不同系统。

- 三界天赋：玩家选择阵营后长期生效，表达阵营身份。
- 动态强弱：根据赛季局势每日调整，表达人数或活跃度修正。

建议 v0.2 首版先实现三界天赋，再接动态强弱叠加。

后续叠加时建议采用：

```text
先算三界天赋，再算动态强弱。
```

并为关键项设置总上限：

- 被掠夺损失降低总上限建议不超过 `35%`。
- 掠夺收益加成总上限建议不超过 `25%`。
- 攻击加成总上限建议不超过 `10%`。

这样可以避免出现“人界弱势时几乎抢不动”或“魔界弱势时重新滚雪球”的极端情况。

## 七、28 天模拟验收目标

### 1. 总价值差距

28 天模拟结束后，三阵营最终总价值差距建议控制在：

```text
15% ~ 25%
```

总价值不只看金币，至少应包含：

- 净金币。
- 灵宠等级。
- 灵宠突破进度。
- 灵宠剩余 HP / 平均战斗可用状态。
- 灵根存量与消耗。
- 灵髓、灵玉存量。
- 普通兽魂、稀有兽魂、传说兽魂存量与消耗。
- 灵宠碎片。
- 被 raid 损失。
- raid 获得收益。
- 洗练条件达成情况。

### 2. 各阵营必须在自身指标领先

人界应领先：

- 农田金币收入。
- 被 raid 后保留收益。
- 净金币稳定性。
- 洗练金币节省量。

仙界应领先：

- 灵根收入。
- 灵宠成长速度。
- 防守战主宠耐久。
- 到达关键等级或突破节点的时间。

魔界应领先：

- 作物周转速度。
- 主动 raid 胜率。
- 主动 raid 收益。
- 灵宠突破兽魂节省量。

### 3. 魔界不能多数核心指标第一

魔界不应同时领先以下多数指标：

- 净金币。
- 灵宠成长。
- 平均防守损失。
- 平均剩余 HP / 可持续战斗状态。
- raid 收益。
- 突破进度。

如果魔界在以上多数指标均第一，说明旧版闭环仍未被打断。

### 4. 关键预警线

模拟报告需要增加以下预警：

- 人界农田额外金币是否小于人界被抢损失增量。
- 人界洗练金币减免是否因为洗练未接入而无法观察。
- 仙界防守 HP 是否让早期仙界过硬。
- 仙界经验优势是否在满级后完全归零。
- 魔界作物成熟时间 `-10%` 是否被固定收菜节奏吞掉。
- 魔界突破兽魂减免是否让魔界养宠中后期追赶过快。
- 魔界前 14 天是否形成资源滚雪球，导致后 14 天不可追。
- 被 raid 玩家是否出现连续多日净收益为负。

## 八、实现优先级

### P0：核心规则与配置字段

优先实现以下九条规则：

- 人界成熟收获金币 `+10%`。
- 人界灵宠洗练金币消耗 `-10%`。
- 人界防守战资源损失 `-20%`，并同步降低攻击方收益。
- 仙界收菜获得灵根 `+10%`。
- 仙界挂机经验 `+10%`。
- 仙界防守战主战灵宠最大 HP `+5%`。
- 魔界作物成熟时间 `-10%`。
- 魔界灵宠突破兽魂消耗 `-10%`，向上取整。
- 魔界主动 raid 攻击 `+6%`，防守不加成，无战后回血。

P0 完成前，不进入 28 天机器人平衡测试。

### P1：模拟与统计口径

P0 完成后补齐：

- 魔界动态成熟时间收菜。
- 洗练策略进入勤奋玩家日常。
- 灵宠突破兽魂节省量统计。
- 攻击方收益与防守方损失一致性校验。
- 阵营三领域指标汇总。
- 总价值计算器。
- 魔界闭环预警。

### P2：玩家表达与后台配置

最后补齐：

- 阵营说明文案。
- 管理后台数值配置项。
- 客户端战报中的减损展示。
- 客户端战斗预览中的主动 raid 加成展示。
- 客户端灵宠突破消耗减免展示。
- 客户端洗练金币减免展示。

## 九、客户端展示原则

玩家侧不展示复杂公式，只展示当前阵营最直观的优势。

推荐表达：

人界：

```text
收成更高，洗练更省金币，被掠夺时损失更少，适合稳定经营。
```

仙界：

```text
灵根更多，灵宠成长更快，防守时主战灵宠气血更稳。
```

魔界：

```text
灵田成熟更快，主动进攻更强，灵宠突破更省兽魂。
```

避免表达：

- 削弱魔界。
- 补偿人界。
- 限制强势阵营。
- 弱势阵营专属补偿。

统一表达为三界天然特性：

- 人界善经营，能守财。
- 仙界善养宠，能守宠。
- 魔界善进攻，节奏更快。

## 十、当前结论

v0.2 的核心调整是：

```text
三界都拥有种田、养宠、战斗三个领域的优势。
魔界保留主动进攻身份，但移除无成本战后回血。
人界补足保产能力，仙界补足防守耐久。
```

建议下一步直接以本文件的 v0.2 参数表作为实现口径。

预期结果：

- 人界在金币经营、防守减损和洗练金币效率上领先。
- 仙界在灵根、经验和防守主宠耐久上领先。
- 魔界在作物周转、主动 raid 胜率和突破兽魂效率上领先。
- 魔界不再同时成为经济、防守、成长和续航的最优解。

只要 28 天最终总价值差距能控制在 `15% ~ 25%` 内，并且各阵营在自身主指标上领先，这组三界 v0.2 规则就可以进入后续数值微调阶段。
