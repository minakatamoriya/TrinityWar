# 2026-05-30 田地生命周期三态清理记录 v0.1

## 1. 当前结论

田地生命周期只保留三个可运转阶段：

1. 成长中 `GROWING`
2. 成熟期 `MATURE`
3. 枯萎期 `WITHERED`

空地 `EMPTY` 和锁定 `LOCKED` 仍然是田地占位状态，但不属于作物生命周期阶段。

## 2. 删除内容

本次清理彻底删除旧的播种阶段：

1. 删除数据库枚举 `FieldStatus.SEEDED`。
2. 删除灵植配置里的 `seed_seconds` 字段。
3. 删除前后端类型里的 `seeded` 展示态。
4. 删除 `stageSeconds.seeded` 这类分阶段时间配置。
5. 新一轮种植后直接进入 `GROWING`，不再经过独立的播种倒计时。

## 3. 时间规则

一轮作物的时间从开始培育时刻计算。

1. 开始培育后立刻进入 `GROWING`。
2. `growthSeconds` 倒计时结束后进入 `MATURE`，可收取。
3. 成熟窗口结束后进入 `WITHERED`，仍可收取枯萎收益。

启灵芽规则固定为：

1. 点击培育后进入 `GROWING`。
2. 10 秒后进入 `MATURE`。
3. 不存在额外播种 20 秒阶段。

## 4. 迁移规则

旧数据库迁移时：

1. 旧 `SEEDED` 田地统一转为 `GROWING`。
2. 旧 `seed_seconds + grow_seconds` 合并为新的 `grow_seconds`。
3. 删除 `seed_seconds` 列。

这样旧账号不会丢进度，新账号也不会再生成播种阶段。
