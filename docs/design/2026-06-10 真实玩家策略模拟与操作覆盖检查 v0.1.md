# 2026-06-10 真实玩家策略模拟与操作覆盖检查 v0.1

## 一、目标

本文件承接：

- `2026-06-10 三界阵营优势价值层级平衡梳理 v0.1`
- `2026-06-08 勤奋玩家赛季模拟 v1 规则与统计口径`
- `2026-06-08 勤奋玩家赛季模拟 v1 单日流程与统计口径`

今天重点检查两件事：

1. `season-sim v1` 是否可以继续模拟更接近真实玩家的日常策略。
2. 好友互助是否应该并入主模拟，以及还有哪些玩家操作尚未纳入。

## 二、当前结论

好友互助应并入 `season-sim v1` 主流程，而不是继续后置到 v1.1。

原因：

- 好友采摘会产生金币与贡献。
- 好友浇水会改变作物成熟时间，间接影响农田节奏。
- 贡献值影响阵营俸禄档位。
- 如果社交完全缺席，28 天模拟会低估勤奋玩家的金币、贡献和农田周转优势。

本轮已采用低复杂度策略接入：

- 每名玩家每天 2 次拜访好友灵田。
- 日初 1 次，优先覆盖好友已有成熟田，主要观察采摘。
- 日末 1 次，农田循环后再拜访，主要观察浇水。
- 单次拜访中仍按真实入口规则：可采摘优先，其次浇水。
- 没有可助力对象时记为 `no-op`，不算卡点。

## 三、真实玩家策略更新

### 1. 原策略

旧单日流程偏向纯主循环：

```text
农田循环
-> 领取俸禄
-> 灵宠成长
-> 跨阵营掠夺
-> 快照
```

这个流程能看种田、养宠、掠夺，但会漏掉真实玩家常见的“顺手处理社交机会”。

### 2. 新策略

当前单日流程调整为：

```text
日初好友互助
-> 农田循环
-> 领取俸禄
-> 灵宠成长/灵根训练加速/突破
-> 日末好友互助
-> 跨阵营掠夺
-> 快照
```

这样安排的好处：

- 日初容易命中初始化或隔夜留下的成熟田，能观察采摘收益。
- 日末更容易命中成长中田地，能观察浇水缩短时间。
- 两次互助足够覆盖核心社交行为，不会把模拟变成互点刷量。

## 四、好友互助统计口径

日报现在应观察：

- `socialHarvestCount`：好友采摘次数。
- `socialWaterCount`：好友浇水次数。
- `socialNoopCount`：无可助力次数。
- `socialGoldIncome`：采摘金币收入。
- `socialShortenedSeconds`：浇水累计缩短秒数。
- `socialIntimacyGain`：好感增长。

后台赛季模拟日报已展示：

- 好友互助：采摘 / 浇水 / 空转。
- 互助收益：金币 / 缩短秒数。
- 阵营维度和玩家维度的互助次数与收益。

## 五、与三界平衡文档的关系

好友互助本身不是阵营优势，但它会改变三界优势的表达环境：

- 人界：采摘金币和浇水加速会放大农田经营节奏，需要确认人界高产出是否因此进一步扩大。
- 仙界：更多贡献与资源流入会影响灵根训练节奏，需要观察仙界成长优势是否过早封顶。
- 魔界：如果掠夺收益仍明显压过社交与农田收益，说明魔界闭环仍强。

因此 28 天报告里不应只看农田、灵宠、掠夺三条线，还要看社交在总金币、贡献和节奏中的占比。

## 六、已纳入主模拟的操作

当前 `season-sim v1` 已覆盖：

- 赛季模拟账号重置与初始化。
- 4 块田初始可用。
- 成熟田收取。
- 空田播种青灵麦。
- 田地按模拟时间结算。
- 日初/日末好友灵田拜访。
- 好友采摘。
- 好友浇水。
- 阵营俸禄领取。
- 灵宠挂机经验结算。
- 灵根训练加速。
- 灵宠突破判断。
- 每日 3 次跨阵营掠夺。
- 掠夺真实结算与防守损失统计。
- 每日玩家快照。
- 每日阵营汇总。
- 动作成功、失败、no-op 与卡点记录。

## 七、代码核对后的已确认缺口

### P0：三界 v0.2 三领域规则尚未接入

当前 `season-sim v1` 已经能跑日循环，但它仍使用旧阵营参数：

- 人界：尚未接入成熟收获金币 `+10%`。
- 人界：尚未接入灵宠洗练金币消耗 `-10%`。
- 人界：尚未接入防守战中资源损失 `-20%`，且攻击方收益也要同步降低。
- 仙界：尚未接入收菜获得灵根 `+10%`。
- 仙界：尚未接入挂机经验 `+10%`。
- 仙界：尚未接入防守战中主战灵宠最大 HP `+5%`。
- 魔界：尚未接入作物成熟时间 `-10%`。
- 魔界：尚未接入灵宠突破兽魂消耗 `-10%`，向上取整。
- 魔界：尚未接入主动进攻攻击 `+6%`，防守不加成，无回血。

因此，当前赛季循环适合验证流程完整性，但还不能作为三界 v0.2 平衡结论的最终依据。

### P0：真实玩家策略仍偏固定

当前模拟采用低复杂度勤奋玩家策略，优点是稳定、可复现，但还没有完全模拟真实玩家的选择行为：

- 好友互助固定拜访环形好友中的下一位，不会扫描所有好友后选择可采摘或可浇水收益最高的目标。
- 掠夺目标已修正为跨阵营，但仍偏固定轮转，不会优先选择当前可打、非保护期、高收益或低风险目标。
- 作物固定为青灵麦，不会根据在线频率、成熟周期、资源需求或阵营优势选择不同作物。
- 魔界作物成熟时间 `-10%` 要求后续模拟按真实成熟时间动态收菜，不能继续只按固定 3 小时、每天 8 轮观察。
- 灵根训练加速目前按资源与加速剩余时间做单次判断，尚未决定是否允许勤奋玩家在一天内多次连续投入灵根。

这些策略不会阻塞第一轮跑通，但会影响后续“真实玩家最优策略”模拟。

### P0：28 天平衡报告仍缺验收层

当前日报能观察玩家和阵营的动作结果，但还需要补三界平衡文档要求的赛季级判断：

- 三阵营最终总价值差距是否落在 `15% ~ 25%`。
- 人界农田额外收益和洗练金币减免是否被 raid 损失抵消。
- 仙界灵根、经验、防守 HP 是否形成清晰但不过量的优势。
- 魔界是否因为成熟时间、主动攻击、突破兽魂减免同时领先过多指标。
- 是否存在玩家连续多日净收益为负。
- 社交金币、社交贡献和浇水缩短时间在总收益中的占比。

没有这层报告时，即使 28 天循环跑完，也只能说明流程可运行，不能直接说明数值已平衡。

## 八、尚未纳入的玩家操作

### P0：影响平衡判断，建议优先纳入或明确排除

1. 灵宠洗练/洗点。

当前文档已标记后置。它会消耗金币、灵髓、灵玉，并影响战斗力与赛季继承价值。若 28 天平衡目标包含词条质量，后续必须接入。

2. 掠夺深度窥视。

真实玩家在掠夺前可能查看目标灵宠信息。它会影响目标选择、天机符消耗和胜率。当前模拟直接按目标池轮转，不做侦察策略。

3. 每日任务领奖已废弃，不再进入 `season-sim`。

当前设定下，玩家完成种田、养宠和战斗等日常行为时直接增加贡献值，不再通过“做任务 -> 领奖”的二段式结构发放额外资源。

后续如果重新引入每日任务领取，也建议把奖励折算进贡献体系或贡献相关结算里，而不是在赛季模拟主循环中新增一条独立经济来源。

4. 赛季签到。

签到会带来周期性奖励。若奖励含灵根、兽魂、灵髓、灵玉或金币，缺席会影响 28 天成长曲线。

### P1：影响真实行为，但可在主平衡稳定后补

1. 作物解锁与多作物选择。

当前统一种青灵麦，适合做基准模拟。但真实玩家可能选择不同成熟周期或资源产出作物。

2. 领地科技/建筑升级。

当前模拟不做科技升级策略。若科技影响收成窗口、灌溉、丰熟或阵营供奉台，会影响长期收益。

3. 刷新掠夺目标。

当前按固定跨阵营目标选择。真实玩家可能刷新目标或挑选更高收益/更低风险目标。

4. 复仇。

战报里存在复仇语义，真实玩家被打后可能优先复仇。当前模拟只做主动跨阵营掠夺。

5. 好友请求、邀请绑定、删好友、恢复关系。

当前模拟直接准备机器人好友关系，不模拟从陌生人到好友的过程。

### P2：商业化、活动或长线行为，暂不进入平衡主循环

1. 天机符购买或广告领取。
2. 公开外链助力。
3. 好友邀请活动奖励。
4. 灵宠商店购买。
5. 通知奖励领取。
6. 后台人工操作。
7. 阵营转移。

这些行为会影响真实留存和商业化，但不适合作为第一版三界基础平衡的核心变量。

## 九、下一步建议

第一步，先用当前版本跑 1 日和 28 日模拟，观察：

- 好友采摘是否主要发生在日初。
- 好友浇水是否主要发生在日末。
- `socialNoopCount` 是否过高。
- 社交金币占总金币比例是否过高。
- 社交贡献是否明显改变俸禄档位。

第二步，再决定是否接入 P0 操作：

- 若灵髓、灵玉大量囤积，优先接洗练。
- 若掠夺胜率过高或过低，优先接深度窥视和目标选择策略。
- 若 28 天资源曲线明显偏低，再接签到。
- 每日任务领奖已不作为当前玩法入口；种田、养宠、战斗的日常价值应直接进入贡献值。

第三步，把三界 v0.2 三领域规则接入模拟后，重点看：

- 人界农田收益加成是否被掠夺抵消。
- 仙界成长优势是否在满级后失效，主战灵宠耐久加成是否能补足长期价值。
- 魔界是否仍同时领先金币、战斗、续航和防守损失。
- 好友互助是否放大了某一阵营原本的优势。

第四步，补赛季级验收报告：

- 输出 28 天最终总价值、人均总价值和阵营差距。
- 输出各阵营主指标是否领先。
- 输出预警线命中情况。
- 输出建议下一轮调参方向。
