# Trinity War 晚入场追赶反滥用规则草案 v0.1

## 一、文档目标

这份文档只解决四个问题：

1. 晚入场追赶机制怎样防止被小号和套利账号滥用。
2. 哪些玩家可以拿追赶 buff 和追赶任务。
3. 追赶账号的赛季奖励资格如何限制，既公平又不劝退。
4. 风控应该卡哪些行为，避免追赶机制变成刷奖励入口。

这份草案的核心原则只有一句话：

追赶机制是给真晚入场玩家补基础盘，不是给老玩家开小号薅赛季红利。

## 二、首发设计底线

首发版本建议守住下面六条：

1. 每个账号每赛季只能完整获得一次追赶资格。
2. 追赶奖励只能补基础盘，不能直通头部榜单收益。
3. 晚入场玩家可以正常玩，但赛季头部奖励需要满足最低参与门槛。
4. 追赶 buff 不能通过删号重建、换端重登反复领取。
5. 异常账号应进入风控池，降低或冻结追赶红利。
6. 风控规则应尽量轻量，不要误伤正常新玩家。

## 三、追赶资格建议

首发建议同时满足“时间条件”与“进度条件”时，才开放完整追赶资格。

建议条件：

1. 账号在本赛季开始若干天后才创建或首次进入该赛季。
2. 当前主城、外场、兵力等核心进度明显低于赛季平均线。

这样可以避免下面两类误判：

1. 老账号故意摆烂后再吃追赶包。
2. 赛季初正常开服玩家也被系统重复发追赶红利。

## 四、一次性与限时规则

首发建议把追赶机制定成“一季一次、限时生效”。

建议规则：

1. 每个账号每赛季只触发一次完整追赶资格。
2. 追赶 buff 只在前几天或前几档核心功能补齐前生效。
3. 追赶任务链完成或到期后自动关闭。
4. 关闭后不能再次重复领取同一赛季追赶包。

这样做的目的很简单：

1. 避免玩家故意反复卡低进度吃包。
2. 保证追赶机制只承担“拉进主循环”的职责。

## 五、赛季奖励资格门槛

这里要明确区分两件事：

1. 游玩参与资格。
2. 高阶赛季结算资格。

首发建议所有玩家都可以随时进入当赛季、正常做任务、正常匹配和正常成长。

限制的不是“能不能玩”，而是“能不能拿完整的高阶赛季结算奖励”。

更稳的做法是区分两层资格：

1. 参与资格。
2. 完整榜单奖励资格。

建议规则：

1. 所有晚入场玩家都可以获得基础参与奖励。
2. 只有满足最低参与天数、最低活跃次数或最低赛季进度后，才开放完整榜单奖励资格。
3. 未达到门槛的玩家，不争头部赛季点券，只拿基础参与型回报。

这条规则的价值是：

1. 真新玩家不会觉得白玩。
2. 老玩家也不会觉得小号晚进来就能蹭满奖励。

## 六、建议的最低门槛类型

首发不需要把门槛做复杂，但需要至少有两道。

建议从下面几类中选两到三类：

1. 最低参与天数。
2. 最低登录次数。
3. 最低主城阶段。
4. 最低任务链完成度。
5. 最低赛季活跃分。

这里不建议只看单一指标，例如只看注册天数或只看登录次数，因为太容易被刷。

## 七、风控建议

首发风控不需要做成重审核，但需要有基础识别。

建议重点关注下面几类行为：

1. 同一账号赛季内重复触发追赶资格。
2. 同设备或同环境短时间创建多个晚入场账号。
3. 账号创建后极短时间内异常完成大量追赶任务。
4. 账号进度异常跳升但正常玩法行为很少。
5. 追赶账号集中冲某类赛季奖励后快速弃坑。

命中风控时，建议处理方式从轻到重：

1. 冻结追赶 buff。
2. 冻结追赶任务奖励发放。
3. 仅保留基础参与奖励。
4. 取消完整赛季结算资格。

## 八、首发推荐的最小规则组合

如果首发要做最小可用版本，建议直接采用下面这套：

1. 每赛季每账号只可完整触发一次追赶资格。
2. 追赶 buff 和追赶任务只限时生效。
3. 晚入场玩家默认可拿基础参与奖励。
4. 头部赛季点券奖励需要满足最低参与门槛。
5. 同设备短时间多账号追赶行为进入风控池。
6. 异常账号只保留基础参与奖励，不给完整追赶红利。

只要守住这六条，追赶机制就能同时满足三件事：

1. 真新玩家能追上主循环。
2. 老玩家不容易用小号套利。
3. 赛季奖励和点券体系不会被追赶机制反咬。
