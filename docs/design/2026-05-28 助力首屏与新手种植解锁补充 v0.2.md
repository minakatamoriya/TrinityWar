# 2026-05-28 助力首屏与新手种植解锁补充 v0.2

## 1. 我的资源分组

“我的资源”需要把资源来源和用途分开展示，避免所有灵宠相关资源混在一起。

当前分组：

1. 灵宠养成材料：灵根、灵髓、灵玉。
2. 兽魂突破材料：普通兽魂、稀有兽魂、传说兽魂。
3. 掠夺精魄：通过掠夺掉落的具体灵宠精魄，例如苍狼精魄、玄虎精魄。
4. 田地精华：已解锁作物对应的精华库存。
5. 其他资源：后续新增的通用资源。

## 2. 新手种植资格

新玩家完成第一次启灵芽种植和收获后，不立即开放青灵麦和风云稻。

正确流程：

1. 新玩家领取启灵芽。
2. 播种并收获启灵芽。
3. 合成首宠。
4. 完成教程掠夺。
5. 回到阵营领取第一次阵营俸禄。
6. 领取成功后，明确提示“青灵麦和风云稻种植资格已解锁”。

如果玩家在领取阵营俸禄前返回田地并点击“开始培育”，前端必须拦截并提示：

```text
先领取阵营俸禄，领取后会解锁青灵麦和风云稻种植资格。
```

服务端建档也必须配合：新玩家默认只解锁启灵芽，青灵麦和风云稻由首次阵营俸禄写入解锁状态。

## 3. 微信助力页首屏

助力页是外链落地页，不是站内说明页。首屏必须足够精简，模拟“砍一刀”式强行动入口。

待助力状态只展示：

1. 醒目标题：例如“青禾道友邀请你帮他浇水培育灵草”。
2. 中间主按钮：例如“帮 TA 浇水”。
3. 底部小字：助力者登录后可领取相应奖励。

助力成功后，如果是新用户，展示转化文案：

```text
你的朋友已在仙界安营扎寨。一起开辟灵田、培养灵宠吗？
马上就能拥有自己的第一块田。
```

按钮进入新手流程。选择阵营时可以提示优先考虑和好友同阵营。

如果是老用户，成功后进入登录并领取奖励流程。
