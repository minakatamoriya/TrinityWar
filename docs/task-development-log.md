先定种子表：每种种子的投入、成熟产出、提前收取折损、成熟时间、是否掉回种子。没有这张表，就没法判断农场是不是主产金口。
再定四周目标表：每周末目标主城、金库、田地位、人口、防守大概到哪一级，然后反推每天应有多少可支配金币。
最后再定消耗曲线：主城和金库负责长期建设消耗，人口和防守负责中后段持续吃钱，造兵和战损恢复负责最后一周把钱重新抽走。
